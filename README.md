# Análisis de Canasta de Mercado - Retail España

Mi primer proyecto de análisis de datos con Python.

## ¿Qué hace?

Analiza 8.000 transacciones de supermercado para encontrar patrones de compra usando el algoritmo Apriori.

## Hallazgos principales

- Vino tinto → Jamón ibérico (49% de confianza)
- Pizza congelada ↔ Helado (55% de confianza)
- Pasta → Aceite de oliva (43% de confianza)

## Tecnologías

- Python 3.x
- Pandas, NumPy
- Matplotlib, Seaborn
- Algoritmo Apriori (implementación manual)

## Cómo usarlo

1. Clona el repositorio
2. Instala dependencias: `pip install -r requirements.txt`
3. Abre el notebook: `jupyter notebook market_basket_analysis.ipynb`

## Datos

Los datos son simulados basados en productos reales de supermercado español. No son datos reales de ninguna cadena.

## Qué aprendí

- Cómo implementar el algoritmo Apriori desde cero
- Que los datos simulados tienen valor si los patrones subyacentes son reales
- Que publicar algo imperfecto es mejor que no publicar nada
